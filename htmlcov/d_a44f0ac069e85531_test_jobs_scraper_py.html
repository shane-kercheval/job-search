39 # Set up mock server using html files in `vercel_clone`; I've copied the files locally so 

40 # we can test on local http-server. If the html ever changes on Vercel's website, consider 

41 # retaining the current function so that we have an example that is tested from local files. 

42 # The main advantage is that we can test changes more quickly (e.g. adding additional fields 

43 # to JobInfo object) compared with hitting external server each time.)
